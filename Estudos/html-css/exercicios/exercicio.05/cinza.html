Desafio de Navegação

Esse é o desafio de navegação do curso de HTML e CSS.

O desafio é criar um site de navegação, onde o usuário possa clicar em links para navegar entre as páginas. O site deve conter pelo menos 3 páginas: Área verde, amarela e cinza.
